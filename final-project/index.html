This full-stack web application addresses a critical public health question: Where are the healthcare access gaps in Los Angeles County, and how can we optimize facility placement to serve underserved communities?

Using advanced geospatial analysis and interactive data visualizations, this platform provides actionable insights for policymakers, healthcare administrators, and community advocates.
