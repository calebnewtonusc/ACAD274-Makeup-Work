Individual Data Visualizations

From LA Healthcare Access Mapping Project

Project Context

These visualizations were created for the LA Healthcare Access Mapping project, analyzing 4,512 healthcare facilities across 2,498 census tracts in LA County serving 9.9 million residents.

Each visualization addresses specific research questions about healthcare accessibility, facility distribution, and underserved communities.

1. Geographic Facility Distribution Map

Location: Analysis → Interactive Maps Section

Description: Interactive choropleth map showing healthcare facility density across LA County census tracts using Leaflet.js and GeoJSON data.

  • Color intensity represents facilities per 10,000 residents
  • Hover tooltips show detailed tract-level statistics
  • KD-tree algorithm calculates nearest facility distances
  • Uses CA State Plane projection (EPSG:2229) for accuracy

Key Finding: South LA and East LA show 2.1 facilities per 10K residents vs. West LA's 7.8 per 10K.

View Live Map →

2. Access Distance Heatmap

Location: Analysis → Heatmap Section

Description: Heatmap showing average distance to nearest healthcare facility by census tract, identifying access deserts.

  • Green areas: <0.5km average distance (good access)
  • Yellow areas: 0.5-1.5km average distance (moderate access)
  • Red areas: >1.5km average distance (limited access)
  • Generated with GeoPandas and Matplotlib

Key Finding: 80,831 residents live in access deserts with >1.5km average distance.

View Live Heatmap →

3. Regional Facility Comparison

Location: Analysis → Regional Breakdown

Description: Interactive bar chart comparing total facilities and per-capita density across LA County regions.

  • Built with D3.js and Chart.js for interactivity
  • Shows both absolute counts and normalized per-capita rates
  • Includes population overlay for context
  • Filterable by facility type

Key Finding: West LA has 2.3x more facilities than South/East LA combined despite smaller population.

View Live Chart →

4. Priority Areas Matrix

Location: Analysis → Priority Matrix

Description: Scatter plot matrix showing relationship between population density, facility count, and socioeconomic indicators.

  • Interactive Plotly visualization with zoom and pan
  • Color-coded by priority level (high, medium, low)
  • Identifies 10 highest-priority census tracts for intervention
  • Incorporates median income and uninsured rate data

Key Finding: 10 census tracts identified as critical priority for new facility placement.
