Chart 2: Distance to Nearest Facility Distribution

Location: Analysis Dashboard → Distance Analysis

Research Question: What percentage of LA County residents live within reasonable distance to healthcare facilities?

Chart Type: Histogram with Cumulative Distribution Function Overlay

Implementation:

  • Generated with Matplotlib and Seaborn for statistical visualization
  • Shows population distribution across distance bins (0-0.5km, 0.5-1km, etc.)
  • Cumulative distribution line overlay shows percentage with access
  • Color gradient from green (good access) to red (poor access)
  • KD-tree algorithm used to calculate nearest facility distances

Chart 3: Healthcare Access vs. Socioeconomic Indicators

Location: Analysis Dashboard → Socioeconomic Analysis

Research Question: How does healthcare access correlate with socioeconomic factors like income and insurance coverage?

Chart Type: Multi-axis Interactive Scatter Plot with Regression Lines

Implementation:

  • Built with Plotly for advanced interactivity (zoom, pan, hover)
  • Each point represents a census tract with color-coded priority level
  • X-axis: Median household income | Y-axis: Facility density per capita
  • Linear regression line shows strong positive correlation
  • Statistical analysis with SciPy and Statsmodels (correlation coefficient, p-value)
